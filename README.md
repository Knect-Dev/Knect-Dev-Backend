### Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo
   ```sh
   git clone https://github.com/Knect-Dev/Knect-Dev-Backend.git
   ```

   ```sh
   npm install i
   ```
4. Enter your API in `config.js`
